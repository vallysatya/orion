# Support

Need help with Orion?

## Self-serve

1. [Documentation index](docs/README.md)
2. [Troubleshooting guide](docs/guides/troubleshooting.md)
3. [CLI reference](docs/reference/cli.md)
4. [Changelog](CHANGELOG.md) / [Roadmap](ROADMAP.md)

Useful local commands:

```bash
orion version
orion doctor
orion config show
orion guard test
```

## Community channels

- **Bug reports / feature requests:** [GitHub Issues](https://github.com/vallysatya/orion/issues)
- **Questions / design discussion:** [GitHub Discussions](https://github.com/vallysatya/orion/discussions)
  (enable/use Discussions if available on the repository)

## What to include in a report

- Orion version (`orion version`)
- Python version (`python --version`)
- OS
- Minimal reproduction steps
- Relevant command output (**redact secrets**)
- Whether `orion doctor` / `orion guard test` pass

## Security vulnerabilities

Do not file public issues. See [SECURITY.md](SECURITY.md).

## Expectations

Orion v1.0 is an open-source project maintained in good faith. Response times
vary. Clear reproductions get faster help.
